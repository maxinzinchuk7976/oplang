Це мінімалістична мова програмування, основною задачею, якої є можливість введенння машинних інструкцій. Нею можна легше вивчати машинні інструкції та досліджувати поведінку комп'ютера. Розроблена особисто **maxinzinchuk_7976**.

## Синтаксис
Мова немає складних конструкцій чи різноманіття форм
```
    %include std/elfheader.op ; імпортуємо шапку ельфа) Необхідно для запуску на Linux
    &мітка 1234 56 78 ; створюємо мітку та вставляємо hex байти після неї.
    *мітка ; це було чотирьох байтне посилання на мітку, є ще 1*мітка та 8*мітка, одно- та восьми- байтні посилання
    %include std/text.op ; імпортуємо методи
    488d3d  *var_str ; закидаємо в регістр rdi, тут без RTFM нікуди
    e8*fn_print ; використовуємо посилання на метод 

    &var_str "Слава РТФ!" 0a 00 ; Рядки теж можна робити
    &var_longstr " Ба більше
    ДУУУУУУУУУУУУУУУУУУУУУУУУЖЕ ; упс, але це не коментар)
    ДОООООООООООООООООООООООВГІ
    "
```
